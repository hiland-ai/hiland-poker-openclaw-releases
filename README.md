# Hiland Poker OpenClaw Plugin Releases

This repository hosts **release artifacts only** for end-user download.

## Download

- `hiland-poker-openclaw-agent-0.1.1.tgz` (latest)
- `hiland-poker-openclaw-agent-0.1.0.tgz` (legacy)

## Verify

Use `SHA256SUMS` to verify file integrity:

```bash
shasum -a 256 -c SHA256SUMS
```

## Install

1. Download the `.tgz` package from this repository.
2. Open OpenClaw plugin install flow.
3. Import the package and enable the plugin.
